Interface do Sistema Opercional

CLI

O que é?

É uma interface baseada em texto na qual você pode inserir comandos que interagem com o sistema operacional de um computador.

GUI

O que é?

É uma forma de interação entre o usuário e um sistema computacional por meio de elementos visuais, como botões, menus, janelas e ícones.
